# hello.trois

Examples using [TroisJS](https://github.com/troisjs/trois) - Vue 3 + Typescript + Vite + TroisJS

## Getting Started

```shell
# Install dependencies
npm install
# Start development mode...
npm run dev
# ...or build for production
npm run build
```
